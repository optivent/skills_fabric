## License Verification (If using an Enterprise License)

**Endpoint:**

`POST beacon.langchain.com/v1/beacon/verify`

**Request:**

```json
{
  "license": "<YOUR_LICENSE_KEY>"
}
```

**Response:**

```json
{
  "token": "Valid JWT" // Short-lived JWT token to avoid repeated license checks
}
```
