## Api Key Verification (If using a LangSmith API Key)

**Endpoint:**
`POST api.smith.langchain.com/auth`

**Request:**

```json
"Headers": {
  X-Api-Key: <YOUR_API_KEY>
}
```

**Response:**

```json
{
  "org_config": {
    "org_id": "3a1c2b6f-4430-4b92-8a5b-79b8b567bbc1",
    ... // Additional organization details
  }
}
```
